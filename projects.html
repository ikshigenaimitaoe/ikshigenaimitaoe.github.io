Airline Ticket Price Prediction

Using XGBoost, AWS SageMaker, and Streamlit

The Need for Airline Ticket Price Prediction

Airline ticket prices fluctuate wildly based on several factors such as demand, seasons, and economic conditions. This unpredictability presents a challenge for both travelers and airlines. To solve this, we developed an AI-based machine learning model that predicts future airline prices with high accuracy, allowing users to make informed booking decisions.

Why XGBoost?

We chose XGBoost for its renowned performance in handling large datasets and structured data. The key reasons include:

  • Exceptional accuracy for complex prediction models.
  • Fast training time with large volumes of data.
  • Handles missing values, noisy data, and outliers effectively.

Model Training with AWS SageMaker

The model was trained on AWS SageMaker, a fully managed machine learning service. SageMaker’s scalability and security allowed us to train the model efficiently on large datasets while optimizing costs. The training process involved several iterations to fine-tune the model for optimal performance.

Deployment with Streamlit

After training the model, it was deployed using Streamlit to provide an interactive web app. This allows users to input flight details and get real-time price predictions with a smooth and intuitive interface.

Challenges and Learnings

Like any AI project, this one presented its share of challenges. We worked through issues such as data quality and ensuring the model generalizes well to unseen data. AWS SageMaker’s infrastructure and advanced features helped us address these issues, resulting in a robust, scalable model.
